# PROJET-GSR
Ce projet a été réalisé en groupe dans le cadre du pôle projet de CentraleSupélec, sous la supervision d'Émilie Avignon-Meseldzija, en réponse à la situation concrète d'un étudiant de l'école souffrant de crises d'épilepsie. L'objectif est de lui proposer un dispositif portable capable d'anticiper ces crises en temps réel grâce au GSR.
